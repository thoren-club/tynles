# Что это за приложение (фиксируем смысл продукта)

## Коротко (1 предложение)
**Это таск‑менеджер и трекер целей с геймификацией (XP/уровни/лиги) для личного и командного использования, работающий как Telegram Mini App.**

## Для кого
- **Для одного человека**: личная дисциплина, привычки, ежедневные дела.
- **Для группы**: семья/друзья/команда — общий прогресс, мотивация через рейтинг.

## Базовые сущности
- **Пространство (Space)**: «контейнер» для задач/целей и людей. Может быть личным или общим.
- **Роли**: Admin / Editor / Viewer (права внутри пространства).
- **Задача (Task)**: действие, которое можно выполнить; бывает одноразовой и повторяющейся.
- **Цель (Goal)**: более крупная штука, которую «закрывают» подтверждением.
- **XP / уровень**: награда за выполнение (прогресс и мотивация).
- **Лиги / лидерборд**: соревнование в рамках периода.
- **Истории (Stories)**: краткие недельные итоги.
- **Напоминания**: эмоциональные уведомления о задачах.

## Главный пользовательский цикл (core loop)
1. Пользователь открывает приложение.
2. Выбирает/создаёт пространство.
3. Добавляет задачи и цели.
4. Выполняет — получает XP, растёт уровень.
5. Видит прогресс, историю недели и позицию в лидерборде.
6. Получает напоминания/«пинки», возвращается в приложение.

## Что важно в UX (как мы будем принимать решения)
- **Минимум трения**: быстро добавить задачу, быстро выполнить.
- **Понятная мотивация**: за что даётся XP, почему так, какой следующий шаг.
- **Честные правила**: лиги/рейтинги и повторяющиеся задачи работают предсказуемо.
- **Приватность**: в командных пространствах видно только то, что должно быть видно.

## Что НЕ является целью (пока)
- Не «супер‑планировщик» с сотней настроек.
- Не корпоративный Jira/Asana.
- Не соцсеть (социалка — только как мотивация, без лишнего шума).
