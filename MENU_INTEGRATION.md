# Интеграция меню с командами

Для полной интеграции всех разделов нужно добавить callback handlers в соответствующие файлы команд.

Это большой объем работы. Рекомендую добавить по частям, тестируя каждый раздел.

Основная идея: заменить текстовые ответы на редактирование сообщения с кнопками меню.
